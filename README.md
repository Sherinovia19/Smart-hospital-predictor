Smart Hospital Resource Predictor

Smart Hospital Resource Predictor is a real-time dashboard that monitors hospital resources and predicts future requirements. It helps hospital administrators make informed decisions by visualizing beds, ICU occupancy, and oxygen usage trends over time.

Live Demo

Access the app here:
https://sherinovia19-smart-hospital-predictor-dashboard-kxcibf.streamlit.app/

Features

Real-time hospital resource monitoring
,Beds occupied
,ICU occupancy
,Oxygen consumption
,Usage trends visualization over time
,Predictive analytics for resource planning
,Interactive, responsive dashboard

Tech Stack

UI & Dashboard: Streamlit
,Data Handling: pandas, numpy
,Visualization: matplotlib, seaborn, plotly
,Deployment: Streamlit Cloud
